# Prueba tecnica2  Consolidación de Órdenes y automatizacion en python 


## Descripción general
Automatización en Python que integra:
- Excel
- API REST
- Web Scraping

## Instalación  de requerimientos en python 3.10 
pip install -r requirements.txt
## Instalación  de requerimientos en otras versiones como  python 3.13 PUEDE NECESITAR CREACION DE ENTORNO VIRTUAL 
## creacion de entorno virtuual 
python3 -m venv venv
## activacion de entorno virtuual 
source venv/bin/activate
## Instalación  de requerimientos 
pip install -r requirements.txt
## Configuración
Copiar .env.example a .env y ajustar valore

## Ejecución
python main.py
