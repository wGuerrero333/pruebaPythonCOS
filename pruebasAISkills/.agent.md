# 🤖 Agente RPA - Procesador de Órdenes

> **Skills Reference**: Ver [SKILL.md](./SKILL.md) para documentación detallada de todas las capacidades.

## 📌 Descripción
Soy un agente inteligente especializado en procesar órdenes de compra de forma automatizada. Mi función es:

1. 📥 **Leer órdenes** desde archivos Excel con validación
2. 🔍 **Validar estado** de cada orden mediante una API REST
3. 🌐 **Obtener información** de libros relacionados mediante web scraping
4. 📊 **Generar reporte** consolidado en Excel con estadísticas

## 🎯 Skills Principales

### 1️⃣ Lectura y Validación de Órdenes
- **Skill**: `cargar_ordenes()`
- Lectura de archivos Excel (`data/ordenes.xlsx`)
- Validación automática de formato y columnas
- Procesamiento iterativo de cada orden
- Registro detallado en logs

**Ver en SKILL.md**: [Lectura y Validación de Órdenes](./SKILL.md#1-lectura-y-validación-de-órdenes)

### 2️⃣ Consulta de Estado en API
- **Skill**: `obtener_estado_api(orden_id, reintentos=3)`
- Endpoint: `https://jsonplaceholder.typicode.com/todos/{orden_id}`
- Estados posibles:
  - **"Aprobada"**: Tarea completada en la API
  - **"En revisión"**: Tarea pendiente
  - **"Error_A"**: Error en la solicitud
- Reintentos automáticos en caso de timeout (máx 3 intentos)
- Backoff exponencial: espera 2 segundos entre reintentos

**Ver en SKILL.md**: [Consulta de Estado en API](./SKILL.md#2-consulta-de-estado-en-api)

### 3️⃣ Web Scraping de Libros
- **Skill**: `obtener_datos_libro(orden_id)`
- Fuente: `https://books.toscrape.com/catalogue/page-1.html`
- Datos extraídos: título, precio, disponibilidad
- Parseo HTML con BeautifulSoup4
- Manejo de errores retorna None

**Ver en SKILL.md**: [Web Scraping de Libros](./SKILL.md#3-web-scraping-de-libros)

### 4️⃣ Generación de Reporte en Excel
- **Skill**: `guardar_resultados()`
- Formato: Excel 2007+ (.xlsx)
- Columnas: `orden_id`, `estado_api`, `titulo_libro`, `precio`, `disponibilidad`
- Salida: `data/reporte_final.xlsx`
- Índices automáticos

**Ver en SKILL.md**: [Generación de Reportes en Excel](./SKILL.md#5-generación-de-reportes-en-excel)


## ⚙️ Configuración

| Parámetro | Valor | Descripción |
|-----------|-------|-------------|
| **API_URL** | `https://jsonplaceholder.typicode.com/todos/` | Endpoint de la API |
| **TIMEOUT** | 5 segundos | Tiempo máximo de espera en conexiones |
| **Reintentos** | 3 intentos | Número máximo de reintentos en API |
| **INPUT_FILE** | `data/ordenes.xlsx` | Archivo de entrada con órdenes |
| **OUTPUT_FILE** | `data/reporte_final.xlsx` | Archivo de salida con reporte |
| **LOG_FILE** | `logs/proceso.log` | Archivo de logs de ejecución |

**Ver en SKILL.md**: [Configuración Completa](./SKILL.md#-configuración)


## 📊 Estadísticas en Tiempo Real

Durante la ejecución, el agente captura:

- **Total de órdenes**: Cantidad cargada desde Excel
- **Órdenes exitosas**: Procesadas sin errores críticos
- **Órdenes con error**: Fallidas o sin datos
- **Desglose de estados**:
  - Aprobadas
  - En revisión
  - Error_A
- **Tasa de éxito**: Porcentaje de órdenes procesadas exitosamente
- **Tiempo total**: Tiempo de ejecución completo

## 🛡️ Manejo de Errores y Recuperación

El agente implementa estrategias robustas de error handling:

| Tipo de Error | Estrategia | Resultado |
|---------------|-----------|----------|
| **Timeout en API** | Reintentos automáticos (max 3) | Reintenta con backoff de 2s |
| **API retorna 5xx** | Reintentos con espera | Estado "Error_A" después de agotar |
| **API retorna 4xx** | Sin reintentos | Estado "Error_A" inmediato |
| **Datos no disponibles** | Retorna None | Continúa con siguiente orden |
| **Archivo no existe** | Validación inicial | Aborta con mensaje claro |
| **Formato Excel incorrecto** | Validación de columnas | Aborta con detalles del error |

**Ver en SKILL.md**: [Validación y Manejo de Errores](./SKILL.md#8-validación-y-manejo-de-errores)
